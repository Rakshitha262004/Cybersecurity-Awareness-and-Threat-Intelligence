# Cybersecurity-Awareness-and-Threat-Intelligence
An AI-driven Threat Intelligence Dashboard that collects, normalizes, and correlates cyber threat data from multiple sources. Provides real-time threat monitoring and visualizations to assist security analysts in proactive decision-making. Built with Python, modular architecture, and database integration.

## 🛠 Project Overview
The Threat Intelligence Dashboard provides real-time monitoring and analysis of potential cyber threats. It collects, normalizes, and correlates threat data to help security analysts make informed decisions.

---

## 🚀 Features
- Real-time threat data collection
- Threat correlation and analysis
- Dashboard visualization
- Modular architecture for easy extension
- Database support for storing threat logs

---

## 📁 Project Structure
| File/Folder        | Description |
|------------------|-------------|
| `app.py`          | Main dashboard application. |
| `model.py`        | ML/AI models for threat analysis. |
| `correlate.py`    | Correlation logic for threat detection. |
| `collectors.py`   | Collects threat intelligence from multiple sources. |
| `normalize.py`    | Normalizes raw threat data. |
| `database/`       | Database scripts and configuration. |
| `screenshots/`    | Dashboard screenshots. |
